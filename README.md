# Math Magicians

> "Math magicians" is a website for all fans of mathematics. It allows users to:
>
> - Make simple calculations.
> - Read a random math-related quote.

## Built With

- React

## Getting Started

- `$ git clone https://github.com/fasaldana/math-magicians.git`
- `$ cd math-magicians`
- `$ npm install`
- `$ npm start`
